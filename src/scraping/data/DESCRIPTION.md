# Lotto 6/49 Historical Dataset

## Overview
Complete historical dataset of all Canadian Lotto 6/49 draws from inception in 1982 to 2025.

## Dataset Details
- **Total Draws**: 3,622
- **Date Range**: June 12, 1982 - November 19, 2025
- **Source**: Official historical records
- **Format**: CSV

## Columns
- `Date`: Draw date (Month Day, Year)
- `Num1` to `Num6`: Main numbers (1-49)
- `Bonus`: Bonus number (1-49)

## Usage
Perfect for:
- Statistical analysis
- Machine learning projects
- Probability studies
- Data visualization

## Update Frequency
Manual updates as new draws occur
